VS Code เป็น code editor ยอดนิยม และ extensions ช่วยเพิ่มประสิทธิภาพการทำงานได้มาก

## Essential Extensions

### 1. **Prettier - Code Formatter**
จัดรูปแบบโค้ดอัตโนมัติให้สวยงาม

### 2. **ESLint**
ตรวจสอบโค้ด JavaScript/TypeScript หา errors

### 3. **Live Server**
รัน local development server พร้อม live reload

### 4. **Auto Rename Tag**
เปลี่ยนชื่อ HTML tag คู่อัตโนมัติ

### 5. **GitLens**
ดู Git history ได้โดยละเอียด

## Web Development

- **Tailwind CSS IntelliSense** - autocomplete สำหรับ Tailwind
- **Path Intellisense** - autocomplete file paths
- **CSS Peek** - ดู CSS definition จากใน HTML

## Productivity

- **Todo Tree** - จัดการ TODO comments
- **Better Comments** - ทำให้ comments สวยงาม
- **Bookmarks** - bookmark บรรทัดโค้ดสำคัญ

## Theme & Icons

- **One Dark Pro** - theme สวยงาม
- **Material Icon Theme** - icons สำหรับไฟล์

ลองติดตั้งและทดลองใช้ดู ช่วยเพิ่มประสิทธิภาพการทำงานได้มาก!
